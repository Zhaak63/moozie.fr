COMMAND PREFIX

The default prefix is 

 +

To find a guilds prefix mention 

 @Moozie#9580

Command Example: 

+help

COMMAND SYNTAX

(Please do not include these when typing commands)

 []

- Optional paramater

 <>

- Required parameter

 < thing1 || thing2 >

- Select an option inside

Add a filter your queue of songs.

Usage: 

filter [ filter ]

Example: 

filter bassboost

Aliases: 

filters

Get information about the currently playing song.

Aliases: 

np

Stop the queue and leave the voice channel.

Resume a paused track.

Pause the current track.

Play a song via a link or name. Currently supporting: YouTube, Spotify, Soundcloud, and Discord MP3's

Usage: 

play < song url || name >

Example: 

play Martin Garrix Animals

Aliases: 

p

Play a playlsit via a link or name. Currently supporting: Youtube and Spotify

Usage: 

play-playlist < playlist url || name >

Example: 

play-playlist EDM

Aliases: 

pp

Skip the rest of the current song.

Loop the queue.
